Welcome to CLD

What is CLD?

CLD stands for Coding Linux Distro and is an Arch based linux distrobution made for coders.

Why Arch based?

Arch is an open source distribution and is lightweight and fast. It also got the AUR (Arch User Repository) and the AUR helper (yay)

What comes pre-installed?

A lot of languages and tools like gcc, make, vim, python, openjdk, rust, vscode, neovim, etc. Pre-installed. Its just a customized Arch linux with pre-installed tools so the user can just install CLD and everything will be ready.

AUR, Arch User Repository

The AUR is a powerfull package manager for Arch. It allows you to install and manage packages from the AUR. The AUR has over 90k packages and the packages are added by users like you which makes it easier to find packages and tools you want to install using the pacman package manager and yay package manager.
